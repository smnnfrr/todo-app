# 🌸 Pink Todo List

A beautiful, minimal, and aesthetic single-page Todo List application built using vanilla JavaScript, HTML5, and CSS3. The interface features a vibrant bubblegum pink canvas, playful rounded typography, and custom styled components.

## ✨ Features
*  Full-screen pastel pink theme with clean, card-free styling.
* **Persistent Storage:** Uses `localStorage` to save your tasks so they don't disappear when you refresh the page.
* **Inline Editing:** Double-click on any task text to easily edit and update it on the fly.
* **Interactive Toggles:** Custom checkboxes with automatic line-through text decorations for completed tasks.
* **Responsive Layout:** Perfectly centered elements using CSS Flexbox for a consistent experience on any screen size.
